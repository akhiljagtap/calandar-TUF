Premium Wall Calendar - Next.js

Interactive, responsive, and premium wall calendar component built for frontend evaluation.

## Key Features

- **Month / Week / Day Views** with smooth Framer Motion animations  
- **Day Range Selection** with continuous gradient and rounded edges  
- **Mini-Month Calendar Sync**: start/end dates sync with main calendar  
- **Drag-and-Drop Events** with shadow + hover lift  
- **Notes Section** persisted via localStorage  
- **Dynamic Hero Image & Gradient**: panel adapts to dominant image colors  
- **Light/Dark Theme Toggle** (default dark, persistent)  
- **Holiday / Sunday Highlight**: red rounded dates  
- Fully **responsive** for desktop and mobile  

---

Tech Stack - 
React 18 / Next.js 13 (App Router)
Tailwind CSS for styling
Framer Motion for animations
LocalStorage for persistence
